False Rule

  • The false rule method is a mathematical method to find a sign change in a function given an Interval.
  • Instead of considering the midpoint of the interval it calculates the mean value that is defined at the intersection of the x-axis and the secant line that results from joining the points of the function at the ends of the interval.
  • Given an interval that belongs to a function, the point of intersection of the x-axis is found with the secant line that joins the points (xi, f (xi).
  • Then (Xv, f (xy) is evaluated in the function f (x).
  • According to the sign of F (Xm) a new interval is generated.
  • if f (xi) f (xm) < 0 then xm is changed by the upper limit xu.
  • This process is repeated until the root is found according to the established tolerance.
  • It may be possible that in the method execution one of the midpoints coincides with a root, in this case f (xm) = 0.
  • •f it is guaranteed that the interval has a sign change and is continuous. We can argue dispersión because we stop the process when the relative or absolute error is less than the tolerance to compare the current value with the previous one, accuracy because the result is close to the root of the function.
