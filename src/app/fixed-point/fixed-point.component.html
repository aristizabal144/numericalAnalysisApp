Fixed point

  • The fixed point method formulates the equation f (x) = 0 and generates an equation of the form x = g (x) that allows finding a value of x that when replaced in g its result is the same, that is, x is unchanged for g, and additionally that f (x) converges to zero.
  • This method belongs to the open methods, generating a sequence of value that is generally faster than interval methods.
  • It does not require an interval that has a root, thus allowing us to determine initial values without having the conditions required by an interval method.
  • We must have a correct initial value or that it is in the margin of the function to know in which value or in which zone will be.
  • To verify if the method works, one must verify the values in an interval [a, b] for which the function g is continuous and it is true that for every value of x in the interval, g (x) also belongs to the same interval.
