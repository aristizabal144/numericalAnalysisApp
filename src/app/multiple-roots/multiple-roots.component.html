Multiple Roots

  • The function must be continuous throughout its interval.
  • The convergence decreases or even the process stops if there is any division by zero.
  • These problems happens with the secant method as well if the function is plane and f(x) and f(x-1) are equally approximate.
  • The multiple roots method can be described as modifications on the Newton method.
  • The method allows solving non-linear functions that contain critical roots (e.g maximums, minimums) in other words, places where the function and its derivative are zero.
