SOR

  • The successive overrelaxation method (SOR) is a method of solving a linear system of equations Ax=b derived by extrapolating the Gauss-Seidel method.
  • This extrapolation takes the form of a weighted average between the previous iterate and the computed Gauss-Seidel iterate successively for each component.
  • w value refers to lambda and its the weighting factor, which depending on its value the behaviour of the gauss-seidel method Will get better or not.
  • If w = 1 we got Gauss-Seidel.
  • If 0 < w < 1 we got sub-relaxation methods (It’s used to get the convergence of some non-convergent systems by Gauss-seidel).
  • If 1 < w < 2 we got over-relaxation methods (It’s used to get the convergence of some convergent systems by Gauss-seidel that are too slow).
